## 方案一：免费方案（推荐入门）

### 准备工作

1. **注册 GitHub 账号**
   - 访问 https://github.com
   - 点击 "Sign up" 注册

2. **安装 Git**（Mac 已自带，打开终端输入 `git --version` 检查）

### 步骤 1：创建代码仓库

1. 登录 GitHub，点击右上角 **+** → **New repository**
2. 填写：
   - Repository name: `my-shop`（仓库名）
   - Description: 我的电商网站
   - 选择 **Public**
   - 勾选 **Add a README file**
3. 点击 **Create repository**

### 步骤 2：上传网站文件

**方法 A：网页上传（简单）**

1. 进入刚创建的仓库页面
2. 点击 **Add file** → **Upload files**
3. 将本地 `电商网站` 文件夹内的所有文件拖入上传
4. 填写提交信息："Initial upload"
5. 点击 **Commit changes**

**方法 B：命令行上传（推荐）**

```bash
# 1. 克隆仓库到本地
git clone https://github.com/你的用户名/my-shop.git

# 2. 进入目录
cd my-shop

# 3. 删除 README（可选）
rm README.md

# 4. 复制网站文件进去
# Mac:
cp -R ~/Desktop/电商网站/* .
# Windows: 手动复制

# 5. 添加所有文件
git add .

# 6. 提交
git commit -m "发布电商网站"

# 7. 推送到 GitHub
git push
```

### 步骤 3：开启 GitHub Pages

1. 进入仓库页面 → 点击 **Settings**
2. 左侧点击 **Pages**
3. 在 **Build and deployment** 下：
   - Source 选择 **Deploy from a branch**
   - Branch 选择 **main**
   - Folder 选择 **/ (root)**
4. 点击 **Save**
5. 等待 1-2 分钟，刷新页面，你会看到类似：
   ```
   Your site is live at https://你的用户名.github.io/my-shop/
   ```

### 步骤 4：自定义域名（可选）

1. 购买域名（见方案二）
2. 在 GitHub Pages 设置中添加你的域名
3. 在域名服务商处添加 DNS 记录：
   - 类型：CNAME
   - 主机记录：www
   - 记录值：你的用户名.github.io

## 方案二：付费方案（专业域名）

### 步骤 1：购买域名

**国内平台**
- 阿里云：https://wanwang.aliyun.com
- 腾讯云：https://dnspod.cn
- 价格：.com 约 ¥50-80/年

**国外平台**
- Namecheap：https://namecheap.com
- Cloudflare：https://cloudflare.com
- 价格：约 $10-15/年

### 步骤 2：购买服务器/托管

| 服务商 | 产品 | 价格 | 特点 |
|--------|------|------|------|
| 阿里云 | 云服务器 ECS | ¥60起/月 | 国内最快、需备案 |
| 腾讯云 | 云服务器 CVM | ¥65起/月 | 国内最快、需备案 |
| Vercel | Pro版 | $20/月 | 免备案、海外快 |
| Netlify | Pro版 | $19/月 | 免备案、海外快 |

**推荐：Vercel（国内可直接访问）**

### 步骤 3：通过 Vercel 部署

1. 注册 Vercel：https://vercel.com
2. 点击 **Add New** → **Project**
3. 选择 **Continue with GitHub**
4. 选择你的电商网站仓库
5. 配置：
   - Framework Preset: **Other**
   - Build Command: （留空）
   - Output Directory: （留空）
6. 点击 **Deploy**
7. 部署完成后，点击 **Domains** 添加你的域名

### 步骤 4：域名解析

在域名购买平台添加 DNS 记录：

| 类型 | 主机记录 | 记录值 |
|------|----------|--------|
| A | @ | 76.76.21.21 |
| CNAME | www | cname.vercel-dns.com |

## 方案三：国内备案方案

> 适用于服务器在国内的情况（如阿里云、腾讯云）

### 备案流程

1. **购买域名和服务器**
   - 阿里云：https://aliyun.com
   - 腾讯云：https://cloud.tencent.com

2. **提交备案申请**
   - 登录对应平台的备案系统
   - 填写网站信息
   - 上传身份证照片
   - 填写核验单（需打印签字）

3. **等待审核**
   - 初审：1-2 个工作日
   - 管局审核：7-20 个工作日（各省不同）

4. **备案成功**
   - 获得备案号
   - 将备案号放在网站底部

### 注意事项

- 个人备案：网站内容需与个人身份相关
- 企业备案：需提供营业执照等
- 备案期间网站不能访问

## 常见问题

### Q1：网站需要HTTPS吗？

**建议添加**：
- Vercel/Netlify 自动提供免费 HTTPS
- GitHub Pages 需手动配置

### Q2：网站有访问量限制吗？

| 平台 | 每月流量限制 |
|------|-------------|
| GitHub Pages | 100GB |
| Vercel 免费版 | 100GB |
| Netlify 免费版 | 100GB |

### Q3：可以同时托管多个网站吗？

可以，创建多个仓库即可。

### Q4：网站更新后如何重新发布？

```bash
# 本地更新后执行
git add .
git commit -m "更新内容"
git push
```

Vercel/GitHub 会自动重新部署。

## 快速开始命令

```bash
# Mac 用户快速部署到 GitHub Pages

# 1. 安装 Homebrew（如果没有）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2. 安装 Git（如果没有）
brew install git

# 3. 配置 Git
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"

# 4. 克隆仓库并部署
cd ~
git clone https://github.com/你的用户名/my-shop.git
cp -R ~/Desktop/电商网站/* ~/my-shop/
cd my-shop
git add .
git commit -m "发布电商网站"
git push
```
